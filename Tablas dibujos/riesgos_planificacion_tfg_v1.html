Tabla 2. Riesgos de planificación del Trabajo de Fin de Grado
ID Riesgo Descripción Prob. Impacto Estrategia de mitigación
RT01
Disponibilidad personal
 Posibilidad de no poder dedicar el tiempo previsto al TFG por motivos académicos, laborales o personales. Media Alto Planificar con márgenes temporales holgados; priorizar tareas críticas y distribuir el trabajo en bloques semanales.
RT02
Herramientas o recursos técnicos
 Imposibilidad temporal de acceder a software, plataformas digitales o recursos técnicos necesarios para el desarrollo del TFG. Media Medio Identificar herramientas alternativas antes de comenzar; mantener copias locales; usar servicios en la nube como respaldo.
RT03
Complejidad metodológica
 Fases metodológicas que resulten más complejas de lo previsto, requiriendo más tiempo y dificultando el ritmo de avance. Alta Alto Dedicar tiempo de estudio a aspectos complejos; replantear profundidad o alcance si se detectan retrasos significativos.
RT04
Carga derivada de revisiones
 Las revisiones del tutor pueden requerir más tiempo del estimado para ser incorporadas. Media Medio Reservar tiempo adicional en cada hito; consultar con el tutor ante dudas; documentar versiones para facilitar ajustes.
RT05
Desviación del enfoque
 Desviarse del objetivo principal añadiendo tareas o contenidos no previstos que aumenten la carga total del TFG. Baja Medio Revisar periódicamente los objetivos y el índice; validar con el tutor cualquier cambio significativo de alcance.
RT06
Retraso en la recopilación de información
 Dificultad para acceder a fuentes fiables, actualizadas o suficientes para fundamentar el trabajo. Media Medio Ampliar la búsqueda a bases de datos académicas secundarias; iniciar la recopilación cuanto antes para disponer de margen.
RT07
Pérdida de información o versiones
 Pérdida accidental de archivos o versiones del trabajo por fallos técnicos, errores humanos o problemas con dispositivos. Baja Alto Realizar copias de seguridad en la nube; usar un sistema de versionado (Git, OneDrive o Google Drive).
RT08
Exceso de extensión
 Dado lo ambicioso del proyecto, exceder los límites de extensión establecidos por la UOC al desarrollar los capítulos. Alta Medio Aplicar disciplina de alcance en cada capítulo; revisar extensión tras cada PEC; volcar el detalle técnico en anexos cuando proceda.
