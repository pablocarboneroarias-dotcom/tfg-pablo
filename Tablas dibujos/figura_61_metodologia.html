🗓 Program Increment (PI) Cada fase del proyecto equivale a un PI de SAFe. Al inicio de cada fase se celebra el PI Planning, donde todos los equipos (funcional, técnico, integraciones) sincronizan objetivos, identifican dependencias y definen el plan del incremento.
🚂 Agile Release Train (ART) Los equipos funcional, técnico e integraciones operan como un único ART coordinado por el Release Train Engineer (RTE), rol asumido por el Project Manager. Las dependencias críticas entre equipos se resuelven en el PI Planning y se monitorizan semanalmente.
🔧 Innovation & Planning Sprint Entre cada fase se reserva un sprint de estabilización e innovación (IP Sprint) para resolver deuda técnica, documentar, preparar el entorno PRE y planificar el siguiente PI antes del go-live. Garantiza que cada fase se cierra limpiamente antes de iniciar la siguiente.
