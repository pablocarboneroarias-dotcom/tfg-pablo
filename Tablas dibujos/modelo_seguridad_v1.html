Capa 2 — Permisos · perfil de mínimo acceso + Permission Set Groups por departamento
Perfil base · Minimum Access – Salesforce

Asignado a todos los usuarios de la org. No contiene permisos funcionales. Solo define login, IP, app por defecto y tipos de registro. Todos los permisos de objeto, campo y funcionalidad se gestionan exclusivamente mediante Permission Set Groups.

ATC → Contratos de Ventas

Los agentes ATC pueden consultar (lectura) los contratos gestionados por el equipo comercial para resolver incidencias y consultas de cliente sin necesidad de elevar su rol.

owner-based · lectura
Contratación → Casos ATC
